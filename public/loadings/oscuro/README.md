# Réplica del icono

Archivos incluidos:
- `index.html` — página que contiene el SVG del icono.
- `styles.css` — estilos y variables de color.

Cómo abrir:
1. Abrir `index.html` en un navegador (doble clic o arrastrar al navegador).

Personalización rápida:
- Cambia los colores en `:root` dentro de `styles.css` (`--white`, `--dark`, `--bg`).
- Ajusta el tamaño modificando la propiedad `.stage` o el ancho de `.logo`.

Notas:
- El icono está hecho con formas SVG (círculos y un rectángulo rotado) para aproximar la imagen original. Si quieres una réplica perfecta, puedo convertir la imagen a paths de SVG precisos.
